# Student Database Analysis (SQL)
## Description
This project uses SQL to analyze student academic data stored in a relational database.
It focuses on retrieving records, calculating averages, filtering data, and performing
department-wise analysis.
## Tools Used
- SQL
- SQLite
## Analysis Performed
- Retrieved student records
- Calculated average marks
- Identified top-performing students
- Filtered students by attendance
- Department-wise performance analysis
